# UI Global Components Directory

This directory will contain generic, reusable UI components from Figma exports.

## Purpose

Generic UI components that can be used across the entire application, such as:
- Buttons
- Cards
- Badges
- Inputs
- Dialogs
- And other shadcn/ui-style components

## Source

Components will be moved here from `src/imports/figma-raw/ui/` after review and organization.

## Integration Status

Currently empty. Components will be integrated from the Figma raw imports directory.
